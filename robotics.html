1 Hardware

1.1 Mounting usb memory stick for data

  1. Check mounted drives with sudo fdisk -l
  2. Mount disk with sudo mount -o uid=pi,gid=pi /dev/sda1 /mnt/flash64 see http://www.modmypi.com/blog/how-to-mount-an-external-hard-drive-on-the-raspberry-pi-raspian
  3. Unmount when done with sudo umount /mnt/flash64
